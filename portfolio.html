Fabienne Pennewaert

This is the first project I did as part of my « front end web development » training at Interface3. This project was presented after the 5 weeks orientation module.

The goal ? Create the website of your choice to implement the various topics covered during these 5 weeks. The success of this project allowed me to access the one-year training.

For this first challenge, I created the online portfolio of Fabienne Pennewaert, a photographer based in Brussels. I respected her sober aesthetic by using monochrome tones (black and white with the smallest touch of red) in order to highlight only her pictures. I am currently pursuing this project on Wordpress so that she can manage her website by herself. Stay tuned …

Jardin Botanique

Here, I re-created the layout in a mobile version of an existing website.

For this exercise, I chose the Botanical Garden of Meise website (which had recently benefited from a facelift).

The purpose of this exercise was to reorganize the content in a more logical way as well as to create a new, more modern visual. I opted for a clean and colorful design, which highlights nature.

Relax & Enjoy

Group work.

The goal? Invent a concept, and create the site for it from A to Z.
Our idea: Create a website that lists places in Brussels to relax.

The website works with a questionnaire where the user can choose an area and then their particular desires of the moment. The website then offers a personalized search according to the choices of the user.

The biggest challenge of this project was to create a database large enough to offer a large choice to the user. We have identified more than 200 places and activities in our database.

Within my group, we mainly focused on the technical aspect and the smooth running of the website. For this project, design was not our priority, that's why it is very simple. But you can find some of my sketches a the end ...

Code Hackadamie

Project realized in group during a « hackathon » of 2 and a half days on the theme "decode and code".

The instructions were to create a platform for people interested in programming and make them want to take training.

With my team of future « back-end developers », « game developer » and « front-end developers » (like me), we have created a website that offers a quiz.

For each question, there is a little bit of theory and a clue. At the end of the quiz, the user is entitled to a word of explanation about our respective courses as well as the links of several training centers in Brussels.
